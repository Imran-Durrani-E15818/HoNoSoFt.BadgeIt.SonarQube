What does it do?

This project generate the badges for SonarQube. You must have configured your appsettings.json of badgeIt in order to make it work. Once the configuration is completed, you will be able to query your project and then generate the badge without requiring the user to log into SonarQube.

In order to generate the badges, we use the service of Shield.io. Please understand that this service can also stop you from requesting badges if you do too many request. In the case you don't want to rely on the online service, there is the possibility of installing the service itself on your server. Please follow this link in order to arrive to the GitHub project. Or if you even more lazy, use a docker image

In the future, maybe the badges JavaScript code will be directly included within this project.
